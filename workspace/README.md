# Vero Atelier Dashboard

A Next.js App Router project for a clothing store with a landing page, login screen, and a dashboard for product management and uploads. Uses TypeScript, Tailwind CSS, and shadcn/ui.

## Pages

- /: Landing page
- /login: Login screen (UI only)
- /shop: Customer storefront
- /product/[id]: Product details
- /dashboard: Product list dashboard
- /dashboard/upload: Product upload form
- /dashboard/notifications: Admin notifications
- /dashboard/settings: Studio settings

## Development

```bash
npm run dev
```
